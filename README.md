# oktibor.com

Personal site for Tibor Ótott-Kovács — [oktibor.com](https://oktibor.com)

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion

## Development

```bash
npm install
npm run dev
```

## Content

All site content lives in `src/data/*.json` — edit those to update the site without touching components.

## Deployment

Deploys to Vercel on push to `main`.
